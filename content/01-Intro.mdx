# Úvod do NextLabs

Ahoj, vítej v NextLabs

## Co je NextLabs?

NextLabs je platforma, která poskytuje `provisioning`. To znamená, že můžete vytvářet své vlastní laby, kde můžete testovat různé technologie a nástroje.

## Nastavení SSH klíče

### Vygenerování SSH klíče

Pokud nemáte SSH klíč, můžete si ho vygenerovat pomocí následujícího příkazu:

```bash
ssh-keygen -t rsa -b 4096
```

Příkaz se vás zeptá, kam chcete klíč uložit. Pokud nechcete měnit cestu, stiskněte Enter.

Poté se vás zeptá na heslo. Můžete ho nechat prázdné.

Klíč se vygeneruje a uloží do `~/.ssh/id_rsa.pub`.

### Nahrání SSH klíče

Pokud už máte SSH klíč, stačí ho jen nahrát.

Na Linuxu nebo MacOS, můžete použít následující příkaz:

```bash
cat ~/.ssh/id_rsa.pub
```

Na Windows klíč najdete v `C:\Users\username\.ssh\id_rsa.pub`.

Klíč vložíte do pole pro nahrání SSH klíče.

## Připojení k VPN

1. Vygenerujte si svůj vpn client v [administraci](/account).

![Account panel](/img/account-panel.webp)

2. Připojte se do VPN sítě pomocí OpenVPN klienta. Pokud nemáte OpenVPN klienta nainstalovaného, můžete si ho stáhnout <a href="https://openvpn.net/client/" target="_blank">zde</a>.
3. Po připojení ověřte, že máte přístup do lab sítě. V prohlížeči zkuste načíst webový server v lab síti <a href="http://10.0.0.10/" target="_blank">zde</a>.

## Vytvoření labu

Pokud jste se úspěšně připojili do VPN, můžete pokračovat vytvořením svého labu.

V administraci si vyberte lab, který chcete vytvořit a klikněte na tlačítko "Start!".

Po chvíli se vám zobrazí informace o vašem labu, včetně IP adresy serveru. Tuto IP adresu budete potřebovat pro připojení k serveru pomocí SSH.

## Připojení k labu pomocí SSH:

Ukážeme si, jak se připojit k serveru pomocí SSH a nastavit webový server Nginx.

```
ssh root@server_ip
```

### Aktualizace apt

Po přihlášení k labu použijte následující příkaz pro aktualizaci apt balíčků:

```
apt update
```

### Instalace Nginx webového serveru

Použijte následující příkaz k instalaci webového serveru Nginx:

```
apt install nginx
```

Spusťte Nginx pomocí následujícího příkazu:

```
nginx
```

### Nastavení vlastní HTML stránky

Použijte následující příkaz pro přesun do adresáře, kde jsou uloženy weby:

```bash
cd /var/www/html
```

Upravte defaultní soubor `index.nginx-debian.html` pomocí textového editoru nano:

```
nano index.nginx-debian.html
```

Pro uložení změn stiskněte `Ctrl + X`. Uložíte změny `Y`. Potvrdíte cestu k souboru `Enter`.

Poté v prohlížeči otevřete IP adresu vašeho serveru a uvidíte vaši novou HTML stránku.
