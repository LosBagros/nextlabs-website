# Remote development

Visual Studio Code umožňuje vzdálený vývoj pomocí SSH tunelu. Tímto způsobem se můžete připojit k vzdálenému serveru a pracovat na svém projektu.

## Instalace Remote Development Extension

1. Otevřete si Visual Studio Code.
2. Klikněte na Extensions.
3. Nainstalujte <a href="https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-ssh" target="_blank">Remote - SSH</a> extension.
4. Command Palette (Ctrl + Shift + P) -> Remote-SSH: Connect to Host... -> Add New SSH Host...
5. Zadejte root@ip_labu a stiskněte Enter.

Za pár sekund se vám otevře nové okno s připojením k vzdálenému serveru.

## Také můžete využít labu `nextlabs:vscode`

Po spuštění labu se spustí vscode server na portu 8080. Stačí otevřít prohlížeč a zadat `http://ip_labu:8080`.
